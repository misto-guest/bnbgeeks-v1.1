# Resources

*Topics of interest and reference materials*

## What goes here
- Topics you're interested in or learning about
- Reference materials, guides, documentation
- Examples: "AI/ML", "Web Development", "Productivity Systems"

## Structure
Create a folder per topic:
```
Resources/
├── topic-name/
│   ├── README.md          # Overview and key concepts
│   ├── links.md           # Useful resources
│   └── notes.md           # Your learning notes
```

## When to use
- You find something interesting but don't have active project
- You want to save something for future reference
- You're learning a new skill or topic
