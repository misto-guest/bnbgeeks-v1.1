# Projects

*Active work with clear goals and deadlines*

## What goes here
- Projects you're actively working on
- Each project has a defined outcome and timeline
- Examples: "Launch transcription app v2", "Migrate to new hosting"

## Structure
Create a folder per project:
```
Projects/
├── project-name/
│   ├── README.md          # Project overview
│   ├── decisions.md       # Key decisions made
│   ├── progress.md        # Current status
│   └── resources/         # Project-specific references
```

## When to move to Archives
- Project is completed
- Project is cancelled/indefinitely paused
- Move entire folder to Archives/ with date suffix
