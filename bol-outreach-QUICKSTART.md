# Bol.com Outreach - Quick Reference

**Production URL:** https://bol-outreach-production.up.railway.app

## Quick Start Guide

1. **Login** (if required)
2. **Seller Discovery** — Search for sellers by keyword
3. **Create Template** — Save outreach message template
4. **Create Campaign** — New outreach campaign
5. **Add Sellers** — Add discovered sellers to campaign
6. **Send Messages** — Execute outreach to all sellers in campaign
7. **Track Results** — Monitor campaign performance

## Features

- ✅ World-class light theme UI
- ✅ AdsPower anti-captcha integration
- ✅ Template management (save/load/delete)
- ✅ Campaign management (create/track/approve)
- ✅ Seller database and discovery
- ✅ Approval queue workflow
- ✅ Analytics dashboard
- ✅ Health monitoring (/api/health)

## Tech Stack

- **Backend:** Node.js + Express
- **Database:** SQLite
- **Frontend:** Vanilla JS + Custom CSS (Stripe/Vercel-quality)
- **Browser Automation:** Puppeteer + AdsPower profiles
