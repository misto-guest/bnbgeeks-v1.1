## ðŸ”„ Customization Examples

### Change Color Scheme
Edit `template.html` CSS variables:
```css
/* Replace terracotta with your color */
--terracotta: #C17767;  /* Change to your accent */
```

Popular accent colors:
- Purple: #9B59B6
- Blue: #3498DB
- Green: #27AE60
- Orange: #E67E22

### Change Typography
Keep the pairing but swap fonts:
```css
/* Swap Crimson Pro for another serif */
--font-serif: 'Playfair Display', Georgia, serif;

/* Swap Montserrat for another sans */
--font-sans: 'Poppins', -apple-system, sans-serif;
```
