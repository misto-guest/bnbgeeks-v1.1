# Clawdbot Browser Relay Extension - Quick Setup

## Installation (2 minutes)

### Step 1: Install Extension
```bash
clawdbot browser extension install
```

### Step 2: Get Extension Path
```bash
clawdbot browser extension path
```

### Step 3: Load in Chrome
1. Open Chrome
2. Go to: `chrome://extensions`
3. Enable "Developer mode" (toggle in top right)
4. Click "Load unpacked"
5. Select the directory from Step 2
6. Pin the extension to your toolbar

---

## How to Use

### Attach a Tab
1. Open any website (e.g., https://getnada.com)
2. Click the Clawdbot extension icon in your toolbar
3. Badge should show **ON** (green)
4. Tab is now attached!

### Detach
Click the extension icon again to detach.

---

## Badge Status Meanings

- **ON** (green): Attached, ready to use ✅
- **...** (yellow): Connecting to relay
- **!** (red): Relay not reachable ❌
