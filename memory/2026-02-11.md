## Technical Notes

### GhostFetch Architecture
- Stealth-first design with "Ghost Protocol" fingerprinting
- Simulates human behavior at hardware level
- Smart scrolling for infinite feeds (X.com optimized)
- Zero-config browser auto-installation
- REST API server model

### Troubleshooting
- Server startup can take 15-30s on first run (browser initialization)
- Timeout errors: Increase timeout or reduce MAX_CONCURRENT_BROWSERS
- Memory issues: Reduce concurrent browsers to 1
- Browser reinstall: `ghostfetch setup`
