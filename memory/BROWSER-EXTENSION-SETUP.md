# Clawdbot Browser Relay Extension - Setup Guide

## Installation

### Chrome Web Store
**URL:** https://chrome.google.com/webstore

### Search for
"Clawdbot Browser Relay" or "Clawdbot"

---

## How to Use

1. **Install** the extension from Chrome Web Store
2. **Navigate** to any website (e.g., https://getnada.com)
3. **Click** the Clawdbot extension icon in Chrome toolbar
4. **Verify** badge shows "ON" (attached)
