Scheduler Setup

To enable automatic scraping, add these cron entries:

Daily (every day at 2 AM): 
0 2 * * * cd /Users/northsea/clawd-dmitry/seo-backlinks-search/query-manager && python3 scheduler.py daily >> logs/cron.log 2>&1
Weekly (every Monday at 2 AM): 
0 2 * * 1 cd /Users/northsea/clawd-dmitry/seo-backlinks-search/query-manager && python3 scheduler.py weekly >> logs/cron.log 2>&1
Monthly (1st of each month at 2 AM): 
0 2 1 * * cd /Users/northsea/clawd-dmitry/seo-backlinks-search/query-manager && python3 scheduler.py monthly >> logs/cron.log 2>&1
üí° Quick Setup: crontab -e and paste the entries above

Environment Variables

Create a .env file in the query-manager directory:

SERPER_API_KEY=your_serper_api_key_here
SECRET_KEY=your_secret_key_here
API_KEY=seo-query-manager-key

API Documentation

Authentication

All API endpoints require authentication via API key:

# Header
X-API-Key: seo-query-manager-key

# Or query parameter
?api_key=seo-query-manager-key

Endpoints

GET /api/queries List all queries
POST /api/queries Add new query
GET /api/results/{query_id} Get latest results
GET /api/results/{query_id}/all Get all historical results
POST /api/results/{query_id}/run Run scrape manually
GET /api/export/{query_id}?format=json|csv Export results
GET /api/logs Get recent logs
GET /api/status Get system status
